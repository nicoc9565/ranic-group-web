# RANIC GROUP CRM — Implementation Plan

> **For agentic workers (Claude en Antigravity):** Ejecutá este plan tarea por tarea. Cada tarea
> termina en un deliverable verificable (build + preview). Hacé commits frecuentes. Los datos de
> dominio (proveedores, blacklist, templates de email, reglas) son textuales y no negociables —
> copialos exactos del plan/spec.

**Goal:** Construir el CRM de RANIC GROUP LLC en `/admin` (Next.js + Firebase) para gestionar
proveedores, follow-ups, generación de emails, blacklist y prospectos de Expo West 2026, con
sincronización en tiempo real entre dos usuarios.

**Architecture:** Next.js App Router + TypeScript + Tailwind. El repo `ranic-web` se reconstruye
de cero. La home `/` es un placeholder; el CRM vive en `/admin` como área autenticada client-side
contra Firebase Auth, con datos en Firestore (`onSnapshot` en tiempo real). La seguridad real son
las reglas de Firestore (`request.auth != null`).

**Tech Stack:** Next.js (App Router), TypeScript, Tailwind CSS, Firebase (Auth + Firestore),
deploy en Vercel.

**Spec de referencia:** `docs/superpowers/specs/2026-06-25-ranic-crm-design.md` (leer entero antes
de empezar).

## Global Constraints

- **Color primario:** verde oliva `#556B2F` (definir como token de Tailwind, ej. `olive`).
- **Idioma:** español en labels de UI; inglés en todo el contenido de emails.
- **Responsive:** sidebar en desktop, bottom nav en mobile.
- **Auth:** solo usuarios autenticados (Nico + César, ambos admin). Sin colección `users`.
- **Reglas de Firestore:** lectura/escritura solo si `request.auth != null`.
- **Env vars** (Vercel y `.env.local`): `NEXT_PUBLIC_FIREBASE_API_KEY`,
  `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN`, `NEXT_PUBLIC_FIREBASE_PROJECT_ID`,
  `NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET`, `NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID`,
  `NEXT_PUBLIC_FIREBASE_APP_ID`.
- **Secuencia de follow-up:** días `[1, 4, 7, 12]` desde `firstContactDate`.
- **Email signature** (siempre, textual):
  ```
  Nicolas Conti
  Managing Member | RANIC GROUP LLC
  nicolas.conti@ranicgroup.com
  www.ranicgroup.com
  +1 (201) 572-1383
  ```
- **Reglas de email (forzar siempre):** primer contacto empieza con "Dear [Name],"; nunca
  mencionar Amazon (usar "online retailer"); nunca incluir EIN/Resale Certificate/Tax ID salvo
  pedido explícito; "recurring monthly orders" en primeros contactos; presentarse como "online
  retailer based in Summit, NJ".

      test("nextFollowUpDate: step 0 → firstContact + 4 días", () => {
        expect(nextFollowUpDate(base)?.toISOString().slice(0,10)).toBe("2026-06-05");
      });
      test("nextFollowUpDate: sin firstContactDate → null", () => {
